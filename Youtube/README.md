# YouTube Trending Video Analytics

## Project Overview
This project analyzes YouTube trending video data to identify patterns in video popularity, audience engagement, and content characteristics. The goal is to understand what factors influence videos to trend by analyzing views, likes, categories, publishing time, and sentiment in video titles.

---

## Objectives
- Clean and preprocess YouTube trending dataset  
- Perform sentiment analysis on video titles  
- Analyze engagement metrics (views, likes, comments)  
- Rank video categories based on popularity  
- Identify trends in publishing time and duration  
- Build an interactive Power BI dashboard  

---

## Dataset
The dataset contains information about YouTube trending videos including:

- Video Title  
- Channel Title  
- Category ID  
- View Count  
- Likes  
- Comment Count  
- Published Date  
- Trending Date  

### Additional Features Created
- Publish Day  
- Publish Month  
- Trending Duration  
- Sentiment Score  
- Sentiment Label (Positive, Neutral, Negative)

## Data Cleaning and Preprocessing
- Removed duplicate records  
- Standardized column names  
- Converted date columns to datetime format  
- Created new time-based features  
- Performed sentiment analysis on video titles  
- Classified titles into Positive, Neutral, and Negative  

---

## Data Analysis
- Category-wise analysis of total views  
- Sentiment distribution of video titles  
- Relationship between views and likes  
- Analysis of trending videos by weekday  
- Trending duration analysis  

---

## Dashboard
An interactive Power BI dashboard was created including:

- Total Videos  
- Total Views and Likes  
- Top Categories by Views  
- Sentiment Distribution  
- Trending Videos by Weekday  
- Trending Duration Distribution  
- Views vs Likes Analysis  

---

## Key Insights
- Most video titles have **neutral sentiment**  
- Certain categories generate **higher view counts**  
- Strong relationship between views and likes  
- Publishing day impacts trending probability  
- Trending duration varies across videos  

---

## Conclusion
This project provides insights into YouTube trending videos by analyzing engagement metrics, content characteristics, and sentiment patterns. The findings help understand viewer behavior and improve content strategies for better reach and engagement.
