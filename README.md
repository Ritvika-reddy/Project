## 🏥 Healthcare Appointment No-Show Prediction

### 📌 Overview
This project analyzes healthcare appointment data to predict whether a patient will miss an appointment. It combines data analysis, machine learning, and Power BI visualization to identify key factors influencing no-shows.

### 🎯 Objectives
- Analyze factors affecting patient attendance  
- Predict no-show behavior using machine learning  
- Handle class imbalance using SMOTE  
- Build an interactive Power BI dashboard  

### 🛠 Tools Used
- Python (Pandas, NumPy)  
- Scikit-learn  
- Matplotlib, Seaborn  
- Power BI  

### 📊 Key Insights
- Around **20% of appointments are missed**  
- Longer waiting time increases no-show probability  
- SMS reminders reduce missed appointments  
- Certain age groups have higher no-show rates

## 📺 YouTube Trending Video Analytics

### 📌 Overview
This project analyzes YouTube trending video data to uncover patterns in video popularity, engagement, and content characteristics using Python, SQL, and Power BI.

### 🎯 Objectives
- Clean and preprocess YouTube dataset  
- Perform sentiment analysis on video titles  
- Analyze engagement metrics (views, likes)  
- Rank video categories using SQL  
- Build an interactive dashboard  

### 🛠 Tools Used
- Python (Pandas, NumPy)  
- Matplotlib, Seaborn  
- TextBlob (Sentiment Analysis)  
- SQL  
- Power BI  

### 📊 Key Insights
- Most video titles have **neutral sentiment**  
- Certain categories generate higher views  
- Strong relationship between views and likes  
- Publishing day influences trending probability
